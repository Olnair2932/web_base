## 📱 O Projeto
Este é um **PWA (Progressive Web App)** de elite, projetado para ser rápido, bonito e funcional. Ele permite que as clientes visualizem o estoque em tempo real e façam encomendas diretamente pelo WhatsApp.

### 🌟 Funcionalidades Principais
- 🎠 **Banner Dinâmico:** Carrossel de fotos de alta qualidade no topo.
- 🛍️ **Catálogo Inteligente:** Lista de produtos com fotos, nomes e preços.
- 💬 **Integração WhatsApp:** Botão de compra que já envia o nome do produto.
- 💖 **Prova Social:** Seção de recadinhos e depoimentos de clientes satisfeitas.
- 🌙 **Modo Noturno:** Conforto visual para navegação noturna.
- 📲 **Instalável:** Funciona como um aplicativo no celular (ocupa quase zero de memória).

---

## 🛠️ Painel Administrativo (Nexus SRE)
O gerenciamento do site é feito inteiramente via linha de comando no **Termux**, garantindo agilidade e controle total sobre o inventário.

| Comando | Função |
| :--- | :--- |
| `./nexus_estoque.sh` | Adiciona novos produtos e atualiza o carrossel |
| `./nexus_editar.sh` | Altera preços de produtos existentes |
| `./nexus_remover.sh` | Remove itens vendidos ou fora de linha |
| `./nexus_comentar.sh` | Adiciona depoimentos de clientes |
| `./check_syntax.sh` | Verifica a saúde e integridade do sistema |
| `./deploy.sh` | Sincroniza todas as alterações com o GitHub |
